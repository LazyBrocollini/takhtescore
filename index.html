Security Architecture

Your browser never writes to GitHub directly. The certification flow:

  1. Frontend (GitHub Pages) — Collects match data, sends a signed POST to the Cloudflare Worker. No tokens exist in the browser.
  2. Cloudflare Worker — Receives the request, reads GITHUB_TOKEN from its secret store, fetches the current data/matches.json, computes hashes, appends the new record, and commits back via the GitHub Contents API.
  3. GitHub Repository — Every certification becomes a git commit. The full audit trail lives in git history, never just a file.
