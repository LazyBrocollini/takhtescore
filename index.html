How certification works ▾

Each match record is certified by computing a SHA-256 digest of its canonical form:

  1. Take all fields except hash
  2. Sort keys alphabetically
  3. Serialize to compact JSON (no extra whitespace)
  4. Compute SHA-256 → store as the hash field

Any change to any field — even a single character — produces a completely different hash, making tampering immediately detectable. Verification runs entirely in your browser using the Web Crypto API.
